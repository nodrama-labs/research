* Description

POC for an aggressive short-mode bear GEM model. Goal: validate whether
ranking tokens by negative momentum and shorting the weakest beats both
buy-and-hold and a naive equal-weight short of the same universe in
2022's confirmed bear market.

Window: May 1 - Dec 31, 2022 (matches gem_bear_models.org).
Universe: 11 Binance USDⓈ-M perps mixing extreme fallers (SOL, AVAX,
DOT, ADA), moderate fallers (BTC, ETH, LINK, MATIC), and
outperformers (BNB, TRX, XMR). ADA was briefly excluded after an
Oct 18-28, 2022 short-squeeze event drove a -11.7% drawdown on the
best config; re-running without ADA cost ~15pp of return for only
~1.4pp of DD relief, showing that ADA's earlier short (Oct 8-18,
+$2355) more than paid for the squeeze. ADA is therefore retained
and the squeeze event is documented in the holdings annotation.

Each stage reports PASS / NO-PASS on a single criterion. No combined
verdict label - the three flags and the underlying numbers are meant
to be read together, not collapsed into KILL / NO-PORT / PORT-TO-RUST.

- Stage 1: best-config Calmar > naive equal-weight short Calmar?
- Stage 2: best-config Calmar > long-only bear specialist (4.6)?
- Stage 3: robust across configs? (>= 1/3 positive Calmar AND a
  dominant fit_window in the top 6, model-agnostic)

def fetch_klines(symbol: str, start_ms: int, end_ms: int) -> pd.DataFrame:

def fetch_funding(symbol: str, start_ms: int, end_ms: int) -> pd.DataFrame:
    """Fetch historical 8h funding rates for one symbol."""
    url = "https://fapi.binance.com/fapi/v1/fundingRate"
    out = []
    cursor = start_ms
    while cursor < end_ms:
        r = requests.get(url, params={
            "symbol": symbol, "startTime": cursor, "endTime": end_ms, "limit": 1000,
        }, timeout=10)
        r.raise_for_status()
        batch = r.json()
        if not batch:
            break
        out.extend(batch)
        last_ts = int(batch[-1]["fundingTime"])
        if last_ts <= cursor:
            break
        cursor = last_ts + 1
        if len(batch) < 1000:
            break
    df = pd.DataFrame(out)
    if df.empty:
        return pd.DataFrame(columns=["token", "timestamp", "funding_rate"])
    df["timestamp"] = df["fundingTime"].astype(int)
    df["funding_rate"] = df["fundingRate"].astype(float)
    df.insert(0, "token", symbol)
    return df[["token", "timestamp", "funding_rate"]]

#+BEGIN_SRC jupyter-python :session llms :kernel llms_kernel :async yes :exports both
def r_squared(y: np.ndarray, y_hat: np.ndarray) -> float:
    ss_total = np.sum((y - np.mean(y)) ** 2)
    ss_residual = np.sum((y - y_hat) ** 2)
    if ss_total < 1e-12:
        return 0.0
    return 1.0 - ss_residual / ss_total

def average_true_range(close: np.ndarray, high: np.ndarray, low: np.ndarray) -> float:
    n = len(close)
    if n < 2:
        return 0.0
    tr_sum = 0.0
    for i in range(1, n):
        tr = max(
            high[i] - low[i],
            abs(high[i] - close[i - 1]),
            abs(low[i] - close[i - 1]),
        )
        tr_sum += tr
    return tr_sum / (n - 1)

def fit_linear_logspace(x, y):
    ln_y = np.log(y)
    X = np.column_stack([np.ones_like(x), x])
    beta, *_ = np.linalg.lstsq(X, ln_y, rcond=None)
    return beta[0], beta[1]

def fit_linear_pricespace(x, y):
    X = np.column_stack([np.ones_like(x), x])
    beta, *_ = np.linalg.lstsq(X, y, rcond=None)
    return beta[0], beta[1]

def exponential_model(x, a0, a1):
    return a0 * np.power(a1, x)

def fit_exponential(x, y, initial):
    try:
        popt, _ = curve_fit(exponential_model, x, y, p0=initial, method='lm', maxfev=5000)
        a0, a1 = popt
        if np.isfinite(a0) and np.isfinite(a1):
            return a0, a1
    except (RuntimeError, ValueError):
        pass
    return initial

@dataclass
class Position:
    token: str
    weight: float = 0.0
    momentum: float = 0.0
    r2: float = 0.0
    a1: float = 0.0
    atr: float = 0.0

def fit_token_exponential(candles: pd.DataFrame, atr_window: int = 15, r2_exponent: float = 1.0):
    if candles.empty:
        return None
    token = candles.iloc[0]["token"]
    x = np.arange(len(candles), dtype=float)
    closes = candles["close"].values
    highs = candles["high"].values
    lows = candles["low"].values
    if np.any(closes <= 0):
        return None
    try:
        b0, b1 = fit_linear_logspace(x, closes)
    except Exception:
        return None
    initial = (np.exp(b0), np.exp(b1))
    a0, a1 = fit_exponential(x, closes, initial)
    y_hat = exponential_model(x, a0, a1)
    r2 = r_squared(closes, y_hat)
    momentum = (r2 ** r2_exponent) * (a1 - 1.0) * 100.0
    lookback = min(len(candles), atr_window)
    mean_close = np.mean(closes)
    atr = (average_true_range(closes[-lookback:], highs[-lookback:], lows[-lookback:]) / mean_close
           if lookback > 1 and mean_close > 0 else 0.0)
    return Position(token=token, momentum=momentum, r2=r2, a1=a1, atr=atr)

def fit_token_linear(candles: pd.DataFrame, atr_window: int = 15, r2_exponent: float = 1.0):
    if candles.empty:
        return None
    token = candles.iloc[0]["token"]
    x = np.arange(len(candles), dtype=float)
    closes = candles["close"].values
    highs = candles["high"].values
    lows = candles["low"].values
    try:
        b0, b1 = fit_linear_pricespace(x, closes)
    except Exception:
        return None
    y_hat = b0 + b1 * x
    r2 = r_squared(closes, y_hat)
    mean_close = np.mean(closes)
    if mean_close <= 0:
        return None
    normalized_slope = b1 / mean_close
    momentum = (r2 ** r2_exponent) * normalized_slope * 100.0
    a1_equiv = 1.0 + normalized_slope
    lookback = min(len(candles), atr_window)
    atr = (average_true_range(closes[-lookback:], highs[-lookback:], lows[-lookback:]) / mean_close
           if lookback > 1 and mean_close > 0 else 0.0)
    return Position(token=token, momentum=momentum, r2=r2, a1=a1_equiv, atr=atr)

#+BEGIN_SRC jupyter-python :session llms :kernel llms_kernel :async yes :exports both
@dataclass
class ShortPosition:
    quantity: float      # positive units of base owed
    entry_price: float

@dataclass
class DaySnapshot:
    total_value: float
    cash: float
    position_pnl: float
    cum_funding_pnl: float
    positions: dict       # token -> current unrealised PnL

class ShortPortfolioState:
    def __init__(self, initial_capital: float):
        self.cash = initial_capital
        self.shorts: dict[str, ShortPosition] = {}
        self.peak_value = initial_capital
        self.cum_funding_pnl = 0.0
        self.history: list[DaySnapshot] = []

    def open_short(self, token: str, price: float, notional: float, fee_rate: float):
        if notional <= 0 or price <= 0:
            return
        fee = notional * fee_rate
        self.cash -= fee
        qty = notional / price
        if token in self.shorts:
            existing = self.shorts[token]
            new_qty = existing.quantity + qty
            new_entry = (existing.entry_price * existing.quantity + price * qty) / new_qty
            self.shorts[token] = ShortPosition(quantity=new_qty, entry_price=new_entry)
        else:
            self.shorts[token] = ShortPosition(quantity=qty, entry_price=price)

    def close_short(self, token: str, price: float, quantity: float, fee_rate: float):
        if token not in self.shorts:
            return
        pos = self.shorts[token]
        qty = min(quantity, pos.quantity)
        pnl = qty * (pos.entry_price - price)
        notional = qty * price
        fee = notional * fee_rate
        self.cash += pnl - fee
        pos.quantity -= qty
        if pos.quantity <= 1e-12:
            del self.shorts[token]

    def apply_funding(self, prices: dict[str, float], daily_rates: dict[str, float]):
        delta = 0.0
        for token, pos in self.shorts.items():
            price = prices.get(token, pos.entry_price)
            rate = daily_rates.get(token, 0.0)
            if not np.isfinite(rate):
                rate = 0.0
            payment = pos.quantity * price * rate
            self.cash -= payment           # negative rate => cash up
            delta -= payment
        self.cum_funding_pnl += delta

    def position_pnl(self, prices: dict[str, float]) -> float:
        total = 0.0
        for token, pos in self.shorts.items():
            price = prices.get(token, pos.entry_price)
            total += pos.quantity * (pos.entry_price - price)
        return total

    def total_value(self, prices: dict[str, float]) -> float:
        return self.cash + self.position_pnl(prices)

    def record_snapshot(self, prices: dict[str, float]):
        total = self.total_value(prices)
        self.peak_value = max(self.peak_value, total)
        per_token = {
            token: pos.quantity * (pos.entry_price - prices.get(token, pos.entry_price))
            for token, pos in self.shorts.items()
        }
        self.history.append(DaySnapshot(
            total_value=total, cash=self.cash,
            position_pnl=self.position_pnl(prices),
            cum_funding_pnl=self.cum_funding_pnl, positions=per_token,
        ))

@dataclass
class GemShortParams:
    r2_threshold: float = 0.5
    momentum_floor: float = 0.02         # require momentum < -momentum_floor
    top_n: int = 3
    fit_window: int = 30
    atr_window: int = 15
    r2_exponent: float = 2.0
    initial_capital: float = 10_000.0
    fee_rate: float = 0.003
    rebalance_cooldown: int = 10

def build_short_portfolio(candidates: list[Position], params: GemShortParams) -> list[Position]:
    filtered = [
        p for p in candidates
        if p.a1 < 1.0
        and p.r2 > params.r2_threshold
        and p.momentum < -params.momentum_floor
        and p.atr > 0.0
    ]
    filtered.sort(key=lambda p: p.momentum)             # ascending -> most negative first
    filtered = filtered[:params.top_n]
    if not filtered:
        return []
    mag_sum = sum(abs(p.momentum) for p in filtered)
    if mag_sum <= 0:
        return []
    for p in filtered:
        p.weight = abs(p.momentum) / mag_sum
    return filtered

def is_in_cooldown(current_day, last_rebalance_day, cooldown_days):
    if last_rebalance_day is None:
        return False
    return cooldown_days > 0 and (current_day - last_rebalance_day) < cooldown_days

def should_rebalance_short(current_shorts: dict[str, float], momentums: dict[str, float],
                           entry_threshold: float, exit_threshold: float) -> bool:
    """
    entry_threshold: e.g. -0.02 (enter when momentum crosses below)
    exit_threshold:  e.g. -0.02 (exit when momentum crosses back above)
    """
    for token, m in momentums.items():
        is_held = token in current_shorts
        if not is_held and m < entry_threshold:
            return True
        if is_held and m > exit_threshold:
            return True
    return False

def rebalance_shorts_to_targets(state: ShortPortfolioState,
                                target_weights: dict[str, float],
                                prices: dict[str, float],
                                fee_rate: float):
    """Differential rebalance: close removed/overweight shorts; open new/underweight shorts."""
    total = state.total_value(prices)
    if total <= 0:
        return
    min_trade = total * 0.001

    # Close anything not in target (or oversized)
    for token in list(state.shorts.keys()):
        price = prices.get(token, state.shorts[token].entry_price)
        current_notional = state.shorts[token].quantity * price
        target_notional = target_weights.get(token, 0.0) * total
        excess = current_notional - target_notional
        if excess > min_trade:
            qty_to_close = excess / price
            state.close_short(token, price, qty_to_close, fee_rate)

    # Re-evaluate total (closes may have moved cash)
    total = state.total_value(prices)

    # Open new or underweight shorts
    for token, w in target_weights.items():
        price = prices.get(token)
        if price is None or price <= 0:
            continue
        target_notional = total * w
        current_notional = (state.shorts[token].quantity * price
                            if token in state.shorts else 0.0)
        gap = target_notional - current_notional
        if gap > min_trade:
            state.open_short(token, price, gap, fee_rate)

def gem_short_backtest(params: GemShortParams,
                       candles_by_token: dict[str, pd.DataFrame],
                       funding_by_token: dict[str, dict[int, float]],
                       fit_fn) -> dict:
    all_timestamps = sorted(set(
        ts for c in candles_by_token.values() for ts in c["timestamp"].values
    ))
    if not all_timestamps:
        return _empty_metrics()

    price_index = {
        token: dict(zip(c["timestamp"].values, c["close"].values))
        for token, c in candles_by_token.items()
    }

    state = ShortPortfolioState(params.initial_capital)
    rebalance_count = 0
    last_rebalance_day = None

    for day_idx, ts in enumerate(all_timestamps):
        prices = {tk: pm[ts] for tk, pm in price_index.items() if ts in pm}
        day_date = pd.Timestamp(ts, unit="ms").floor("D")
        day_ms = int(day_date.value // 10**6)
        daily_rates = {tk: funding_by_token.get(tk, {}).get(day_ms, 0.0) for tk in price_index}

        # 1. Apply funding to currently-open shorts before any decisions
        state.apply_funding(prices, daily_rates)

        # 2. Fit each token using up-to-current candles, trimmed to fit_window
        fit_results = {}
        for token, candles in candles_by_token.items():
            up_to = candles[candles["timestamp"] <= ts]
            if params.fit_window > 0 and len(up_to) > params.fit_window:
                up_to = up_to.iloc[-params.fit_window:]
            if len(up_to) >= 2:
                pos = fit_fn(up_to, params.atr_window, params.r2_exponent)
                if pos is not None:
                    fit_results[token] = pos

        # 3. Rebalance decision
        need_rebalance = False
        if day_idx == 0:
            need_rebalance = True
        elif not is_in_cooldown(day_idx, last_rebalance_day, params.rebalance_cooldown):
            momentums = {t: p.momentum for t, p in fit_results.items()}
            holdings = {t: pos.quantity for t, pos in state.shorts.items()}
            need_rebalance = should_rebalance_short(
                holdings, momentums,
                entry_threshold=-params.momentum_floor,
                exit_threshold=-params.momentum_floor,
            )

        if need_rebalance:
            portfolio = build_short_portfolio(list(fit_results.values()), params)
            target_weights = {p.token: p.weight for p in portfolio}
            rebalance_shorts_to_targets(state, target_weights, prices, params.fee_rate)
            rebalance_count += 1
            last_rebalance_day = day_idx

        state.record_snapshot(prices)

    return _compute_metrics(state, rebalance_count, all_timestamps)

def _empty_metrics():
    return dict(total_return_pct=0.0, annualized_return_pct=0.0, max_drawdown_pct=0.0,
                sharpe_ratio=0.0, sortino_ratio=0.0, calmar_ratio=0.0,
                rebalance_count=0, win_rate_pct=0.0, final_value=0.0,
                price_return_pct=0.0, funding_return_pct=0.0, equity_curve=[])

def _compute_metrics(state: ShortPortfolioState, rebalance_count: int, timestamps: list) -> dict:
    history = state.history
    if len(history) < 2:
        return _empty_metrics()
    values = np.array([s.total_value for s in history])
    initial_value = values[0] if values[0] > 0 else INITIAL_CAPITAL
    final_value = values[-1]
    total_return = (final_value - initial_value) / initial_value
    days = len(history)
    years = days / 365.0
    annualized_return = (1.0 + total_return) ** (1.0 / years) - 1.0 if 1.0 + total_return > 0 else -1.0
    peak = np.maximum.accumulate(values)
    drawdowns = np.where(peak > 0, (peak - values) / peak, 0.0)
    max_drawdown = float(np.max(drawdowns))
    daily_returns = np.diff(values) / values[:-1]
    mean_r = np.mean(daily_returns)
    std_r = np.std(daily_returns)
    sharpe = (mean_r / std_r * np.sqrt(252)) if std_r > 1e-12 else 0.0
    downside = daily_returns[daily_returns < 0]
    downside_dev = (np.sqrt(np.sum(downside ** 2) / len(daily_returns))
                    if len(daily_returns) > 0 else 0.0)
    sortino = (mean_r / downside_dev * np.sqrt(252)) if downside_dev > 1e-12 else 0.0
    calmar = (annualized_return / max_drawdown) if max_drawdown > 1e-12 else 0.0
    monthly_returns = []
    chunk = min(30, len(values))
    for i in range(0, len(values), chunk):
        c = values[i:i+chunk]
        if len(c) > 1:
            monthly_returns.append((c[-1] - c[0]) / c[0])
    win_rate = (sum(1 for r in monthly_returns if r > 0) / len(monthly_returns) * 100
                if monthly_returns else 0.0)
    funding_return = state.cum_funding_pnl / initial_value * 100
    price_return = total_return * 100 - funding_return
    dates = pd.to_datetime(np.array(timestamps[:len(values)]), unit="ms")
    return dict(
        total_return_pct=total_return*100, annualized_return_pct=annualized_return*100,
        max_drawdown_pct=-max_drawdown*100, sharpe_ratio=sharpe, sortino_ratio=sortino,
        calmar_ratio=calmar, rebalance_count=rebalance_count, win_rate_pct=win_rate,
        final_value=final_value,
        price_return_pct=price_return, funding_return_pct=funding_return,
        equity_curve=list(zip(dates, values)),
    )

def print_metrics(label, m):

    print(f"  Total return:      {m['total_return_pct']:>8.2f}%")

def buy_and_hold(candles_by_token, initial_capital=INITIAL_CAPITAL, fee_rate=FEE_RATE):
    """Equal-weight long across all tokens, held start-to-finish."""
    timestamps = sorted(set(ts for c in candles_by_token.values() for ts in c["timestamp"].values))
    price_index = {t: dict(zip(c["timestamp"].values, c["close"].values))
                   for t, c in candles_by_token.items()}
    tokens = [t for t in candles_by_token if timestamps[0] in price_index[t]]
    w = 1.0 / len(tokens)
    # Open positions on day 0 - track unitwise
    units = {}
    cash = initial_capital
    for t in tokens:
        notional = initial_capital * w
        fee = notional * fee_rate
        cash -= notional + fee
        units[t] = notional / price_index[t][timestamps[0]]
    history = []
    cum_funding = 0.0
    for ts in timestamps:
        equity = cash + sum(units[t] * price_index[t].get(ts, 0.0) for t in tokens)
        history.append(DaySnapshot(total_value=equity, cash=cash,
                                   position_pnl=equity-cash, cum_funding_pnl=0.0,
                                   positions={}))
    state = ShortPortfolioState(initial_capital)
    state.history = history
    return _compute_metrics(state, 0, timestamps)

def naive_equal_weight_short(candles_by_token, funding_by_token,
                             initial_capital=INITIAL_CAPITAL, fee_rate=FEE_RATE):
    """Equal-weight short across all tokens on day 0, never rebalanced. With funding."""
    timestamps = sorted(set(ts for c in candles_by_token.values() for ts in c["timestamp"].values))
    price_index = {t: dict(zip(c["timestamp"].values, c["close"].values))
                   for t, c in candles_by_token.items()}
    tokens = [t for t in candles_by_token if timestamps[0] in price_index[t]]
    state = ShortPortfolioState(initial_capital)
    w = 1.0 / len(tokens)
    prices0 = {t: price_index[t][timestamps[0]] for t in tokens}
    for t in tokens:
        state.open_short(t, prices0[t], initial_capital * w, fee_rate)
    for ts in timestamps:
        prices = {t: price_index[t].get(ts, state.shorts[t].entry_price if t in state.shorts else 0)
                  for t in tokens}
        day_date = pd.Timestamp(ts, unit="ms").floor("D")
        day_ms = int(day_date.value // 10**6)
        daily_rates = {t: funding_by_token.get(t, {}).get(day_ms, 0.0) for t in tokens}
        state.apply_funding(prices, daily_rates)
        state.record_snapshot(prices)
    return _compute_metrics(state, 1, timestamps)

* Follow-up: larger universe re-run

The 10-token universe used here was deliberately small for a fast POC.
What this run cannot tell us:

- *Does the signal scale to a portfolio?* At top_n>=3 the model's
  Calmar drops below the long-only-bear benchmark on this universe,
  but with only 7 fallers available, n>=3 is forced to dilute into
  weaker signals. A larger candidate pool is a confound, not a result.

- *Is naive's Stage 1 advantage structural or specific?* Naive's lower
  max drawdown comes from holding the entire 10-token universe
  equally; a single-asset short is mathematically unable to match that
  DD floor. Whether the gap survives on a 25-35 token universe with
  top_n=5 is the question that decides whether this is a real
  portfolio strategy or a concentration trade.

** Proposed follow-up POC

Re-run the same notebook with these changes:

1. *Expand universe to 25-35 Binance USDⓈ-M majors that existed in May
   2022.* Use exchange-info to filter by listing date. Reject tokens
   delisted before Dec 31, 2022.

2. *Re-derive LONG_ONLY_BEAR_CALMAR on the new universe* before any
   Stage 2 comparison. The 4.6 reference number is universe-specific
   to gem_bear_models.org and is meaningless on a different universe.

3. *Focus the sweep on top_n in {3, 5, 8}* with fit_window=15. The
   single-asset configs already worked; the open question is whether
   diversification at scale retains the return while shrinking the DD.

4. *Pre-register the diagnostic checks before reading results:*
   - Does naive's Calmar still beat the best config?
   - Does top_n>=3 retain >= 70% of the n=1 return?
   - Does the holdings trace show fewer CASH stretches with a wider
     candidate pool (more chances of a momentum cross)?

** What would change the recommendation

- *PORT-grade signal:* if naive Calmar <= best Calmar AND top_n>=3
  retains return AND DD shrinks, the strategy is real portfolio-grade.

- *Concentration trade only:* if n=1 still wins after widening the
  universe but n>=3 collapses, the model is a "find the biggest
  faller" oracle, not a portfolio strategy. Useful as a signal feed
  into a discretionary process, not as a Rust autotrader.

- *Universe-fragile:* if the lin-n1-w15 win does not reproduce on the
  wider universe at all, this POC's result was a small-sample
  artifact and the track should be dropped.
